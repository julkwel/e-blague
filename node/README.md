## FB CLI HOW TO USE
Use Command to login :

``` bash
fb --login
```

# Navigation

``` bash
fb --navigate /me
```
Or
``` bash
fb --navigate /me/posts
```
to get all group posts
``` bash
fb --navigate /{group-id}/feed
```

to get feed from E-Blague
``` bash
fb --eblague
```
